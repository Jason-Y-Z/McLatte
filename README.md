# McLatte

Multi-Cause LAtent facTor iTe Estimator

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install mclatte.

```bash
pip install mclatte
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
